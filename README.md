# TravelEase — Transport Booking System

A full-stack transport booking web app built with Node.js, Express, MongoDB, and vanilla HTML/CSS/JS.

## 🚀 Live Demo
Deploy link :
[https://travelease-iql4.onrender.com](https://travelease-iql4.onrender.com)

## 🏗️ Tech Stack
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT (JSON Web Tokens)
- **Payment:** Razorpay
- **Deployment:** Render

## 📁 Project Structure

TravelEase/
├── backend/
│   ├── server.js          # Main Express server
│   ├── models/            # Mongoose models
│   ├── routes/            # API routes
│   ├── middleware/        # Auth middleware
│   └── package.json
└── frontend/
    ├── pages/             # HTML pages
    ├── styles/            # CSS
    └── js/utils.js        # Shared JS utilities

## 🚀 Features
- Search buses, trains, flights, cars, autos & bikes
- Seat selection
- Razorpay payment integration
- User dashboard & booking history
- Admin panel
